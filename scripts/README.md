# Analysis scripts

This folder contains R scripts used to perform the analyses presented in the manuscript.

The scripts are organized according to spatial scale (micro and meso) and ecological group (troglobionts and non-troglobionts).

In this folder you can find:
- `Mesoscale_NT_Santana.R`: analyses for non-troglobitic species at mesoscale;
- `Mesoscale_T_Santana.R`: analyses for troglobitic species at mesoscale;
- `Microscale_NT_Santana.R`: analyses for non-troglobitic species at microscale;
- `Microscale_T_Santana.R`: analyses for troglobitic species at microscale.
