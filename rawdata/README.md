# Raw data

This folder contains the raw datasets used in the analyses presented in the manuscript.

The files are provided in their original format, as obtained from field surveys and subsequent data compilation steps. 

In this folder you can find: 
- `Community_Data_Santana.xlsx`: data on the subterranean invertebrate communities on both micro- and mesoscale;
- `Habitat_Mesoscale_Santana.csv`: data on the invertebrate's habitat on mesoscale;
- `Habitat_Microscale_Santana.csv`: data on the invertebrate's habitat on microscale;
- `Mesoscale_UTM.csv`: geographic positioning of the transects;
- `Microscale_UTM.csv`: geographic positioning of the quadracts.
